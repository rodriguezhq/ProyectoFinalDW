Desarrollar un proyecto de un sistema académico integral, implementado con frontend en React + Vite
y backend en Python Flask o Java Spring Boot, que permita gestionar de manera segura y eficiente los
procesos  académicos  de  facultades  y  especialidades.  El  sistema  debe  incluir  módulos  de  matrícula,
cursos  y  docentes,  control  de  notas,  record  académico,  certificados  y  documentos,  así  como
administración  y  seguridad  con  roles  diferenciados  para  estudiantes,  docentes,  administradores  y
dirección.  El  objetivo  es  construir  una  aplicación  funcional,  modular  y  escalable  que  refleje  las
necesidades  reales  de  una  institución  académica  y  demuestre  buenas  prácticas  de  programación  y
trabajo colaborativo.

Módulos a Implementar

1.  Módulo de Matrícula

o  Estudiante: Solicita matrícula, descarga ficha.

o  Administrador: Valida requisitos, registra pagos, genera ficha oficial.

o  Dirección: Supervisa estadísticas de matrícula.

2.  Módulo de Cursos y Docentes

o  Docente: Visualiza cursos asignados, carga sílabos y notas.

o  Administrador: Asigna docentes y gestiona horarios.

o  Dirección: Evalúa carga docente y cumplimiento del plan de estudios.

3.  Módulo de Notas

o  Docente: Registra notas parciales y finales.

o  Estudiante: Consulta hoja de notas por ciclo.

o  Administrador: Valida actas y consolida notas.

o  Dirección: Supervisa indicadores académicos.

4.  Módulo de Record Académico

o  Estudiante: Accede a su historial completo.

o  Administrador: Genera reportes consolidados.

o  Dirección: Analiza desempeño por cohorte o programa.

5.  Módulo de Certificados y Documentos
